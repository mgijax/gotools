MGI GO Slim Categories

The MGI GO_Slim set is defined using unions and disjunctions of GO terms, so that the MGI GO_Slim cannot currently be used with the standard tools (e.g. map2slim script). However, a DAG-Edit readable file is available for download here.

Tab-delimited file mapping GO_ids to MGI GO_Slim category available for download here.
File of genes symbols grouped by GO_id available for download here.

RULES
  • "OR" means the union of the genes that are annotated to the terms
  • "EXCLUDING" means the union of all the genes annotated to the parent term and the other terms of that branch of the vocabulary, except the named term
