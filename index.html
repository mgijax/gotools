MGI Gene Ontology Tools

This page provides links to several MGI specific GO tools.

The ontology and MGI gene annotation files are updated regularly.

Assessing GO annotation consistency
We have developed a new method for assessing the consistency of Gene Ontology [GO] annotations among different GO annotation groups. Our approach utilizes the hierarchical structure of the GO to standardize the existing sets of GO annotations of gene products. The method compares these annotation sets using an independent determination of orthologous gene pairs between organisms. Here we report the results when using this method to compare mouse and human gene annotations to the molecular function GO according to a GO_Slim categorization.

MGI Gene Ontology Term Finder

This GO Term Finder tool searches for significant shared GO terms, or parents of the GO terms, used to describe the genes in your list to help you discover what the genes may have in common.

MGI Gene Ontology GO_Slim Chart Tool

This GO_Slim Chart Tool bins the genes in your list according to MGI GO Slim definitions to help you discover what the genes may have in common.
